position: static;

По умолчанию все элементы имеют position: static;. Это значение явно указывать не обязательно. Такие элементы находяться в потоке и занимают определеное пространство в документе. Свойства top: ...; left: ...; right ...; bottom: ...; никак не влияют на такой элемент

p1 - position: static;
c1 - position: absolute;

position: relative;

Элементы с position: relative; остаються в потоке и занимают определенное пространство в документе. Могут изменять свое положение относительно первоначальной позиции с помощью свойств top: ..., left: ...; right ...; bottom ...; Их используют для размещения маленьких обьектов относительно других на странице.

position: relative; top: 20px; left: 50px;
c2 - position: absolute;

position: absolute;

Элементы с position: absolute; выпадают из потока и разподагаються в заданной позиции относительно ближайшего родителя с position: relative | absolute | fixed. Если такого родителя нету, то элемент позиционируеться относительно body

p3 - position: absolute; right: 20px; left 50px;
c3 - position: absolute;

position: fixed;

Элементы с position: fixed выпадают из потока и фиксируют свое положение относителько экрана (окно браузера). Управлять их положением можно с помощью свойств top: ...; left: ...; right: ...; bottom: ...;

p4 - position: fixed; bottom: 20px; left: 30px;
c4 - position: absolute;
